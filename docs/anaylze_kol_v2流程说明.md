# KOL 数据分析流程文档 (v2)

## 概述

`anaylze_kol_v2.py` 是 KOL 达人评分分析工具 v2，采用**多维度加权评分**方案，综合评估达人投放价值。

**特点**：
- IQR 极值判断（更科学的极值检测）
- 8个评分维度
- 动态 CPM 定价（根据粉丝体量和趋势）
- 可视化输出

## 核心函数

### 1. `clean_play_data()` - IQR 极值处理

使用 IQR（四分位距）方法检测和去除极值。

```
极值判断公式：
  上界 = Q3 + 3 × IQR
  下界 = Q1 - 3 × IQR

  IQR = Q3 - Q1
```

**规则**：
- 超出 `[下界, 上界]` 范围的数据视为极值
- 最多删除2个极值，避免过度清洗
- 负数和空值会被忽略

**返回**：
| 索引 | 含义 |
|------|------|
| 0 | 平均播放（含极值） |
| 1 | 平均播放（清洗后） |
| 2 | 中位数播放 |
| 3 | 是否有爆款 |
| 4 | 爆款数量 |

---

### 2. `get_trend_score()` - 播放趋势判断

使用**线性回归斜率 + 首尾比率**双重判断。

```
趋势判断逻辑：

  1. 计算线性拟合斜率 k
  2. 计算首尾比率 = 最后一个值 / 第一个值
  3. 综合判断：

     上升?  k>0 且 ratio≥1.3  → strong_up (+2)
     上升?  k>0               → mild_up (+1)
     下降?  k<0 且 ratio≤0.7  → strong_down (-2)
     下降?  k<0               → mild_down (-1)
     其他                       → stable (0)
```

**返回**：
```python
("strong_up", 2)   # 强势上升
("mild_up", 1)     # 温和上升
("stable", 0)      # 稳定
("mild_down", -1)  # 温和下降
("strong_down", -2) # 强势下降
```

---

### 3. `dynamic_price()` - 动态 CPM 定价

根据粉丝体量和趋势动态调整 CPM。

```
粉丝量级定价：

  < 5000 粉     → CPM = 8
  5000-30000    → CPM = 15 (或16.5如果播粉比>15)
  30000-100000  → CPM = 18
  > 100000      → CPM = 19.5

趋势溢价：
  上升趋势 → CPM × 1.1

报价计算：
  中位价 = 播放量 × CPM / 1000
  低价   = 中位价 × 0.75
  高价   = 中位价 × 1.25
```

---

### 4. `calculate_total_score()` - 最终评分计算

8个维度综合评分：

```
总分 = 体量分 + 播粉比分 + 稳定性分 + 爆款分 + 性价比分 + 趋势分 + 内容匹配分 + 互动分
```

**评分构成明细**：

| # | 维度 | 最高分 | 判断条件 |
|---|------|--------|---------|
| 1 | 体量 | +6 | 中粉爆款(5k-3w粉+播粉比>15) |
| 2 | 播粉比 | +1 | ≥3 |
| 3 | 稳定性 | +2 | 变异系数<30% |
| 4 | 爆款 | +2 | ≥2个极值 |
| 5 | 性价比 | +2 | CPM<8 |
| 6 | 趋势 | ±2 | 根据趋势类型 |
| 7 | 内容匹配 | ±1 | 完全/部分/不匹配 |
| 8 | 互动率 | ±1 | ≥5%/≥2%/<1% |

**最大总分**：约 **19分**

---

### 5. `calc_interact_rate()` - 互动率计算

```
互动率 = (点赞 + 评论 + 收藏) / 播放量

每个视频独立计算后取平均值
```

---

### 6. `visualize()` - 可视化输出

生成两张图：
- `评分分布.png` - 达人评分分布直方图
- `TOP20达人.png` - TOP20 优质达人条形图

## 主流程 `run_kol_analysis()`

```
输入: CSV文件
  │
  ├─ 1. 读取CSV
  │
  ├─ 2. 清洗播放数据 → clean_play_data()
  │     ├─ 计算含极值平均值
  │     ├─ 计算清洗后平均值
  │     ├─ 检测爆款
  │     └─ 统计爆款数量
  │
  ├─ 3. 计算核心指标
  │     ├─ 播粉比 = 清洗后平均 / 粉丝数
  │     ├─ 变异系数 = std / mean × 100
  │     ├─ 互动率 = (点赞+评论+收藏) / 播放
  │     └─ 实际CPM = 报价 / 清洗后平均 × 1000
  │
  ├─ 4. 趋势分析 → get_trend_score()
  │
  ├─ 5. 动态定价 → dynamic_price()
  │     ├─ 建议底价
  │     ├─ 建议中位价
  │     └─ 建议高价
  │
  ├─ 6. 最终评分 → calculate_total_score()
  │     └─ 输出总分 + 评分构成明细
  │
  ├─ 7. 排序并标注优先级
  │     ├─ 高: 总分≥12
  │     ├─ 中: 总分8-12
  │     └─ 低: 总分<8
  │
  ├─ 8. 输出Excel + 可视化图表
  │
  └─ 返回 DataFrame
```

## 数据指标说明

### 播粉比
```
播粉比 = 稳定播放量 / 粉丝数
```
- > 15: 🚀爆款潜力
- 5-15: 正常偏高
- 1-5: 正常范围
- < 1: 需要关注

### 变异系数 (CV)
```
CV = 标准差 / 平均值 × 100%
```
- < 30%: ✓极稳定 (+2)
- 30-50%: ○较稳定 (+1)
- 50-80%: 正常
- > 80%: ⚠️波动较大

### 互动率
```
互动率 = (点赞 + 评论 + 收藏) / 播放量
```
- ≥ 5%: ✓高互动 (+2)
- 2-5%: ○中互动 (+1)
- < 2%: ⚠️低互动 (-1)

### 实际 CPM
```
实际CPM = 报价 / 清洗后平均播放 × 1000
```
- < 8: 💰超值 (+2)
- 8-15: 正常
- > 15: 偏贵
